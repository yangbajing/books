# PostgreSQL 高级实战

真实项目中的 PostgreSQL 实战应用。

--------
## 基础篇

- 入门，安装
- 入门，SQL
- 日常管理

## 实战应用

- 业务数据建模
- 外部表
- JSON 数据
- 全文检索
- 时序数据

## 高级主题

- 集群：主备
  - 同步复制 quaro
  - 异步复制
- 读写分离：在业务服务中通过多数据源实现读写分离，使用 CQRS 模式。相对搭建专门的读/写分离中间件服务，配置更简单
- CDC：Change Data Capture，变更数据捕获。对于微服务架构，通过 CDC 来实现数据同步，以及做为实时数仓的数据同频机制
